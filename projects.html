bassad

A script to generate a nice list of video files, with links within a directory. I honestly cannot remember the origin of the name, it has just kind of "stuck".

Within the tarball, you will find an index.php file - simply move the file into a directory which will be served by your webserver (typically /var/www/). There are some options you can set within the file: title name, video directory and background color.

hackAttempts.sh

Don't get excited, this is only a reporting tool! It seems like botnets/scriptkiddies like to run scripted attacks against IP addresses with port 22 open. I wanted to keep track of these attacks and find out where they are coming from, so I made my hackAttempts.sh script.

Yes, I realize that I could just change my port number to be protected from these attacks, but that is no fun! Additionally. I use fail2ban to actually stop these scripted attacks.

Please note that you will be required to create a cronjob or a crontab entry if you want this script to run regularly. You will also need to ensure that the 'mail' command is configured and available if you would like the script to email you when there is a new attack.

The download contains the bash script, as well as a sample output log.

Download: hackAttempts.tar.bz2
