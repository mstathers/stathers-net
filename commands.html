Useful Linux Commands

Please note that I typically use Ubuntu, so these commands may not work for other operating systems.
sudo update-alternatives --config editor
Changes default editor

File Managemnet

ls -lh
Lists directory contents with human readable filesizes (eg. KB, MB, etc.)
ls -lt
Lists directory contents ordered based on last modified time
ls -ltr
Lists directory contents ordered based on last modified time(reverse order)
cd -
Toggle change directory between last two directories
find -iname [searchFile]
Searches for file (not case sensitive)
tar xvf [archiveName]
Extract files from a .tar archive

Networking

netstat -r
Display kernel routing table
netstat -i
List network interfaces
netstat -ie
Extended list of network interfaces (alt. ifconfig)
dhclient -v
Refresh DHCP address
ifconfig [interfaceName] down
Disables an interface
ifconfig [interfaceName] up
Enables an interface
ifconfig [interfaceName] [ipAddress] netmask [subnetMask] broadcast [broadcastAddress]
Configures an ip address, subnet mask and broadcast address on an interface. You can choose to exclude any of the mentioned
ifconfig [interfaceName] promisc
Places an interface in promiscuous mode
ifconfig [interfaceName] -promisc
Places an interface in normal mode

Package management

apt-get install [packageName1] [packageName2] [...]
Installs specified package(s)
apt-cache search [packageName]
Searches for a package name based on name or description
apt-cache show [packageName]
Displays basic information about a package
apt-cache showpkg [packageName]
Displays detailed information about a package
apt-cache depends [packageName]
Lists all dependent packages
apt-cache list [packageName]
Lists all files located within a package
dpkg -i [packageName]
Installs a a debian ".deb" package
dpkg -r [packageName]
Uninstalls a ".deb" package

User Management

adduser [username]
Create a user interactively
useradd [username]
Create a user with the defaults as shown by "useradd -D"
passwd [username]
Change password for a user

uname -a
Displays important system information
lshw | less

lshw -short

System Troubleshooting

service --status-all
Check the status of all services
dmesg | less
View boot messages
dmesg | grep [search term]
Search status of different devices.
eg. "dmesg | grep memory" will show memory use, available, total memory and a various other things related to memory
ps -efH | less
Lists currently running processes in a tree structure

Hard Disk Management

fdisk -l
View all available partitions
fdisk -l [diskName]
View all available partitions of a specific disk
fdisk [diskName]
Enter interactive mode (push "m" for help)
